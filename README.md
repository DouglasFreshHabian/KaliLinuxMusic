# 🎧 Kali Linux Music

> *Music for hackers built by hackers, in the terminal.*

---

## 🐉 Overview

**Kali Linux Music** is a curated playlist designed for those who live in the command line.

This project is about more than music, it's about atmosphere.
Late nights. Quiet rooms. Terminal windows glowing.
Every track is meant to enhance deep focus, creativity, and that unmistakable **Linux mindset**.

Whether you're:

* Reverse engineering firmware
* Writing Bash scripts
* Auditing systems
* Or just vibing in your terminal

This is your soundtrack.

---

## ⚡ Usage

Clone the repo and play your tracks however you like:

```bash
git clone https://github.com/DouglasFreshHabian/KaliLinuxMusic.git
cd KaliLinuxMusic
```

Example:

```bash
mpv Kali_Linux_Dreaming.mp3
```

Play Every Song:

```bash
mpv *
```
